Class TowerPlacementGhost

Tower placement "ghost" that indicates the position of the tower to be placed and its validity for placement. This is built with mouse in mind for testing, but it should be possible to abstract a lot of this into a child class for the purposes of a touch UI.

Should exist on its own layer to ensure best placement.
